WRC Pricing Tables is a WordPress Responsive CSS3 plugin which allows you to display pricing tables on different posts or pages by SHORTCODE.


== Description == 
WordPress Responsive CSS3 Pricing Tables has been created to display pricing tables on your WordPress site. It is responsive, beautiful and very easy to use.

There are no limitations! Countless tables can be created and displayed individually on different posts or pages by SHORTCODE. You can also modify each table and their packages at any time. The packages are sortable and each package can be designed by different colors by the ColorPicker. You can also set ribbons and tooltips on each package. Ribbons and link buttons can also be designed by different colors by the ColorPicker. Package features are also sortable, means you can change the position of each feature at any time.


= Features =

* Responsive For All Devices
* Easy To Use
* Unlimited Background Colors
* Unlimited Rows
* Unlimited Columns
* Sortable Columns & Feature Items
* CSS3 Tooltips
* CSS3 Ribbons (Unlimited Colors)
* Tick / Cross Icon Supported
* Implement by Shortcode
* All Major Browser Supported

== Installation ==

Installing WRC Pricing Tables Plugin is as simple as installing any other WordPress Plugin

1. Log In as an Admin on your WordPress blog.
2. In the menu displayed on the left, there is a "Plugins" tab. Click it.
3. Now click "Add New".
4. There, you have the buttons: "Upload Plugin". Click it.
5. Upload the wrc-pricing-tables.zip file and click Install Now button.
6. After the installation is finished, click Activate Plugin.
7. Go to: Pricing Tables >> Guide for the instruction.
